10. 객체
- window : 웹 브라우저 안에 있는 최상위 "객체"
> 내장객체 : window 기본적으로 탑재되어있는 객체

~ window 자주 사용하는 프로퍼디 ~
- window.document / 웹 브라우저에 출력될 문서 => html > body
- innerHeight / 문서 내용의 높이
- innerWidth / 문서 내용의 너비
- outerHeight / 웹 브라우저 창의 높이
- outerWidth / 웹 브라우저 창의 너비
- screenX / 모니터의 왼쪽부터 떨어진 지점의 x좌표
- screenY/ 모니터의 상단부터 떨어진 지점의 y좌표
- scrollX = pageXOffset / 스크롤 시, 수평으로 이동한 만큼의 X 좌표
- scrollY = pageYOffset / 스크롤 시, 수직으로 이동한 만큼의 Y 좌표
- localStorage / 웹 브라우저에서 생성되는 데이터를 저장할 수 있도록 해주는 프로퍼티 (* 웹 브라우저가 종료되어도 데이터 유지)
- sessionStorage / 웹 브라우저에서 생성되는 데이터를 저장할 수 있도록 해주는 프로퍼티(*웹 브라우저가 종료/새로고침 되면 리셋)
- location / window 객체의 현 위치를 URL로 표기

~ window 자주 사용하는 매서드(*함수) ~
-alert( ) / 웹 브라우저 알림창을 표시할 때
- prompt( ) / 사용자로부터 입력된 텍스트를 사용할 때
- focus( ) / 특정 윈도우창에 마우스 커서가 위치했을 때 
- open( ) / 새로운 창을 열고자 할 때
- close( ) / 현재 열려있는 창을 닫고자 할 때 
- confirm( ) / 확인 및 취소 버튼이 있는 확인창을 표시
- print( ) / 현재 웹 브라우저 문서를 출력하고자 할 때
- scroll( ) / 문서에서 특정 위치로 스크롤하고자 할 때
- scrollTo( ) / 문서에서 지정한 위치까지 한 번에 스크롤
- stop( ) / 웹 문서 로딩을 멈추게 하고 싶을 때


- Date( )
~ 프로토타입을 통해서 생성되는 인스턴스 객체가 사용할 수 있는 메서드( )
- getFullYear( ) / 연도값을 가져올 때 : - set~연도 값을 설정할 때
- getMonth( ) / 월 값을 가져올 때 (*0~11 숫자 : - set~ 0 => 1월) : 월 값을 설정할 때
- getDate( ) / 일 값을 가져올 때 (*1~31까지) : - set~ 일 값을 설정할 때
- getDay( ) / 요일 값을 가져올 때 (*0~6까지 / 0=> 일요일) : - set~ 요일값을 설정할 때
- getTime( ) / tlrks rkqtdmf rkwudhf Eo (*1970년 1월 1일부터 현시점까지의 시간 / 밀리초) : - set~ 시간 값을 설정할 때
- getHours( ) / 시 값 : - set~ 시 값을 설정
- getMinutes( ) / 분 값 : - set~ 분 값을 설정
- getSeconds( ) / 초 값 : - set~ 초 값을 설정
