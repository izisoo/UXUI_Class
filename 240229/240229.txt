12. 문자열 & 배열
- charAt( ) : 매개변수의 인덱스값을 가지고 있는 문자열을 찾아올 때
(*문자열[인덱스값]으로도 사용가능!

- indexOf( ) : 매개변수 문자열을  처음 찾았을 때의 인덱스 값을 찾아올 때

- startWith( ) : 문자열 전체에서 처음 시작하는 단어가 맞는지 확인 (*매개변수 2개를 전달 : 첫번째 매개변수가 두번째 매개변수의

- endWith( ) : s 문법과 동일하게 작용 (*문자열의 뒤에서부터 서칭)

- includes( ) : 문자열 안에 매개변수의 문자가 있으면 true / 아니면 false

- trim() / trimStart() / trimEnd() : 문자열의 전체/ 앞/ 뒤에서 빈문자열을 찾아서 없애주는 기능

- uppercase / lowercase

- substring( ) : 매개변수는 1개 혹은 2개 입력 가능
> 매개변수 1개 : 해당 매개변수의 인덱스값부터 문자열의 끝까지 모두 찾아오는 기능
> 매개변수 2개 : 첫번째 매개변수 부터 두번째 매개변수의 바로 앞까지 문자열을 찾아온느 기능
